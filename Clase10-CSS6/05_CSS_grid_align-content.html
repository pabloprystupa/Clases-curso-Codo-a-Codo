CSS Grid | Propiedad align-content

La propiedad align-content se utiliza para alinear verticalmente toda la cuadrícula dentro del contenedor.

Nota: la altura total de la cuadrícula debe ser menor que la correspondiente al contenedor para que la propiedad align-content tenga algún efecto.
