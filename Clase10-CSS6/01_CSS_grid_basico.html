CSS GRID Básico

Esta es la configuración típica de Grid.

La propiedad grid-template-columns define cantidad y tamaño de las columnas.

La propiedad grid-template-rows define el tamaño de las filas.

Hay que tener en cuenta que estas propiedades interactuan tanto con width como con height.

Por supuesto, ademas de px se pueden utilizar otras medidas (%, vw, mm, etc). La unidad especial de Grid CSS fr (fraction) simboliza una fracción de espacio restante en el grid.
